# **App Name**: Kairos Visuals

## Core Features:

- Home Dashboard: Provides a summary view of key operational metrics, including weekly task progress, user workload, critical task alerts, and mobile push notifications via Make integration for critical alerts.
- Calendar Module: Displays tasks and deadlines in monthly and weekly views, with color-coded tasks based on task type configurations. Allows direct task creation by clicking on a date.
- Quotation Generator: Enables the creation of client quotations, including searching existing clients or adding new prospect data, selecting predefined packs or manual tasks, setting quotation validity, and converting approved quotations into projects with automated client and task creation.
- Automated Invoicing: Automates the invoicing process based on monthly pack subscriptions or independent services, with options for tax toggling and integration with consumed extras. Toggle of smart taxation via GenAI Tool: determines whether the type of client or products merit special tax breaks
- Client Management (CRM): Centralizes client information with contact details, asset links (Drive/Dropbox), pack progress tracking, and special pricing tables for tasks outside the pack.
- Reporting and Analytics: Generates financial and productivity reports with filters for date and client, summarizing total amounts and tracking delivery times and workload per user.
- Expense and Credit Tracking: Tracks expenses such as licenses and payments, as well as credit purchases with payment schedules, displaying total paid vs. remaining and alerting due dates.
- Backlog and Kanban Management: Facilitates sprint planning with a task manager, sprint initiation, and visual management of the production flow across Pending, In Progress, Client Review, and Deliverable columns. Automatically returns incomplete tasks to the backlog with high priority.
- Configuration Management: Allows comprehensive customization, user profile creation with role assignments, task type management with color coding, pack definition with task quantities and pricing, and app settings for PWA configuration and webhook setup with Make.

## Style Guidelines:

- Primary color: Deep, saturated purple (#6A29EA) to evoke feelings of insight and precision.
- Background color: Very light purple (#F2E9FF) with a desaturation around 20%, providing a soft contrast.
- Accent color: Dark blue-purple (#2932EA), differing greatly in brightness to provide sharp contrast, drawing focus on details.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, and 'Inter' (sans-serif) for body text.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clean, modern icons that are easily recognizable, with a focus on representing functionality and status clearly.
- Mobile-first design approach with large buttons and bottom menus for easy one-handed use. Consistent color-coding for task types across the calendar, Kanban, and backlog views.
- Subtle animations and transitions to enhance user experience, providing feedback and guiding the user through interactions.
